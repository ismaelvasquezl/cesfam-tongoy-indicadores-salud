# CESFAM Tongoy Indicadores de Salud

Repositorio público con infografías e indicadores de salud del CESFAM Tongoy en formato web, diseñados para visualización clara, acceso móvil y comunicación pública.

## Propósito

Este proyecto busca facilitar la revisión de indicadores sanitarios mediante infografías web responsivas, legibles en computador y teléfono, y orientadas a equipos de salud, gestión local y comunidad.

## Contenido

El repositorio incluye:

- Infografías HTML de corte mensual.
- Visualizaciones comparativas entre periodos.
- Página de inicio (`index.html`) para acceso centralizado.
- Recursos estáticos como estilos, scripts e imágenes.

## Objetivos

- Mejorar la comunicación de indicadores de salud.
- Facilitar el acceso desde dispositivos móviles mediante QR.
- Favorecer la lectura clara de datos para uso interno y comunitario.
- Mantener una presentación visual simple, accesible y actualizable.

## Uso

1. Abrir `index.html` como punto de entrada.
2. Seleccionar la infografía deseada.
3. Visualizar desde computador o celular.
4. Compartir mediante enlace directo o código QR.

## Publicación

Este repositorio está pensado para desplegarse como sitio estático gratuito, por ejemplo en GitHub Pages o Cloudflare Pages.

## Público objetivo

- Equipos de atención primaria de salud.
- Referentes técnicos y administrativos.
- Apoyo a difusión local de indicadores.
- Personas interesadas en visualización clara de información sanitaria.
