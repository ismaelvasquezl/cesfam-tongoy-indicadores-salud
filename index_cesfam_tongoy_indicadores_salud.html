Puntos técnicos incluidos

Dejé el archivo en una sola pieza para que sea más simple de subir a GitHub Pages, pero con bloques ordenados y comentados para mantener buena lectura técnica.

Analítica medible

Los botones usan la clase .track-click y envían eventos a GA4 para saber qué sección recibe más visitas o clics.

Estructura accesible

Incluye skip link, jerarquía semántica, foco visible, botones táctiles de 44 px y soporte para usuarios con reducción de movimiento.

Publicación simple

Subes el archivo como index.html a tu repositorio y GitHub Pages lo mostrará como portada sin pasos adicionales de compilación.
